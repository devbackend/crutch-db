==

Not production ready: making just for fun

Development
--

```shell
# install linters
make lint-install

# run linters
make lint

# run tests (without coverage)
make test

# run tests with coverage
make test-coverage

# build project (local)
make build

# run project (local)
make run

# build project (in docker)
make docker-build

# run project (in docker)
make docker-run
```
